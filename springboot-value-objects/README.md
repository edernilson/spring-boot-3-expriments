# Object values usage and tests

This is a example of usage of object values in a Spring Boot 3 application.

## Stack

* Spring Boot 3.4.0
* Spring Web, Data Jpa,
* Integration and unit tests
* Java 17

## Running the application

1. Running application

    ```shell
    mvn spring-boot:run
    ```

2. Running tests

    ```shell
    mvn spring-boot:test
    ```
